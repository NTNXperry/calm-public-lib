# Ansible Blueprint

This blueprint creates an instance and installs Ansible Server.

##Infra
 1.AWS
 2.Azure
 3.Openstack

##OS
 1. Ubuntu 

## Quick Start
 1. Upload the blueprint to calm.
 2. Change Aws provider in overview tab.
 3. Change Credentials, under credentials tab.
 4. Deploy the blueprint.
